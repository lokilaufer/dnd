# DnD — Drag and Drop доска как Trello

[![Build status](https://github.com/lokilaufer/dnd/actions/workflows/deploy.yml/badge.svg)](https://github.com/lokilaufer/dnd/actions/workflows/deploy.yml)
[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-green?style=flat&logo=github)](https://lokilaufer.github.io/dnd/)

## Возможности

- Три колонки: To Do, In Progress, Done
- Добавление карточек через кнопку
- Удаление карточек (крестик при наведении)
- Перетаскивание карточек внутри колонки и между колонками
- Сохранение состояния в LocalStorage
- Drag with offset (карточка тащится за то место, где схватили)
- Визуальный placeholder при перетаскивании

## Технологии

- Webpack
- Yarn
- GitHub Actions
- LocalStorage API
- Drag & Drop API
